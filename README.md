# Xque

## 为什么会开发 Xque

Xque 是为了让 Xhear 有个更小的 `$` 依赖库而设计的；

Xque 是由 es6 开发的兼容 jQuery库，体积更小，只有 10kb （压缩混淆，gzip能更小）；
